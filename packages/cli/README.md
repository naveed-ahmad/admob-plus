# Usage
<!-- usage -->
```sh-session
$ npm install -g admob-plus
$ admob-plus COMMAND
running command...
$ admob-plus (-v|--version|version)
admob-plus/0.15.8 darwin-x64 node-v15.2.1
$ admob-plus --help [COMMAND]
USAGE
  $ admob-plus COMMAND
...
```

## `admob-plus help [COMMAND]`

```
USAGE
  $ admob-plus help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v3.2.0/src/commands/help.ts)_

## `admob-plus info`

```
USAGE
  $ admob-plus info

OPTIONS
  -h, --help   show CLI help
  --clipboard  copy the environment report output to the clipboard

EXAMPLE
  $ admob-plus info
```

_See code: [src/commands/info.ts](https://github.com/admob-plus/admob-plus/blob/v0.15.8/src/commands/info.ts)_
